# 银杏神枪手 · 成电70年

电子科技大学 70 周年校庆主题的网页射击小游戏：在沙河与清水河校区地图中游园射击，收集校庆回忆与成电 AI 前沿目标。

> 这是校友共建的非官方创作项目，不代表电子科技大学官方立场。

## 共建方式

### 为什么做这个游戏

2026 年，电子科技大学迎来建校 70 周年。很多校友对母校的记忆，也许是一条银杏大道、一栋教学楼、操场边的路灯，或是与同学匆匆走过的某段路。我们把这些零散而珍贵的记忆，做成一场可以回到校园里“打一场”的轻量小游戏：瞄准、命中、收集、分享，再把新的故事带回来。

它不是学校官方项目，也不试图替代严谨的校史与校园档案；更希望成为一个开放的起点——让每一届校友都能给这张数字校园地图添上一笔。

### 当前地图：已有内容与边界

游戏已分别制作沙河、清水河两个独立场景，并使用建筑名称牌和校园氛围化建模呈现游园路线。

- 沙河校区：包含 **211 主楼**、图书馆、第一/第二教学楼、逸夫楼、成电会堂、一九六五楼、软件学院楼、体育馆、食堂、东苑/北苑宿舍、润新公寓（万人坑）等场景线索。
- 清水河校区：包含 **主楼**、图书馆、品学楼、立人楼、科研楼、创新中心、学生活动中心、体育馆、银杏园食堂、商业街、蓝花楹苑、硕博苑、综合训练馆等场景线索。
- 学院标签已覆盖：信息与通信工程学院、电子科学与工程学院、材料与能源学院、机械与电气工程学院、光电科学与工程学院、自动化工程学院、资源与环境学院、计算机科学与工程学院（网络空间安全学院）、信息与软件工程学院（示范性软件学院）、航空航天学院、数学科学学院、物理学院、医学院、生命科学与技术学院、经济与管理学院、公共管理学院、外国语学院。

**当前不足：** 地图是基于公开资料、照片和记忆进行的风格化创作，并非学校授权的权威平面图或厘米级实景复刻；建筑位置、外立面、院系驻地及历史名称仍可能存在偏差。特别欢迎熟悉两个校区的校友补充可靠来源、旧照片、地标故事与校对建议。

### 当前目标物：已有内容与边界

除校庆 70 周年金色气球、普通气球、银杏果和游园主题目标外，游戏已加入六类 AI 前沿目标：

- 光模块：信息与通信工程、光电科学与工程方向
- HBM / 先进封装：电子科学与工程、材料与能源方向
- GPU：计算机、软件工程方向
- 大模型：计算机、自动化、数学方向
- 锂电池：材料与能源、机械与电气方向
- AI + 生物医疗：医学院、生命科学与技术方向

命中后会展示相关学院方向和部分校友企业线索，目的是让“打中一个目标”成为认识成电前沿的一次小入口。

**当前不足：** 目标物的造型、出现频率、文字播报、院系关联和校友企业资料还不完整，部分内容仍需更权威的公开来源校验；玩法平衡、命中反馈、移动端适配、无障碍体验和音效表现也都有持续优化空间。

### 期待校友一起补全

- 校正两个校区的建筑、路线、学院与历史资料
- 提供可授权使用的校园照片、声音、故事与老物件记忆
- 设计新的目标物、关卡、成就、角色、海报和文案
- 改进手机端操作、性能、兼容性、无障碍与多语言体验
- 为 AI 目标补充可核验的专业资料、优秀校友和校友企业线索
## 校友共建与署名

这是一份为校友开放的共建项目。每一位被合并的 Pull Request，都会在 GitHub 的 **Contributors** 页面保留对应 GitHub 账号署名；对被采纳的地图资料、素材、策划、测试和设计建议，我们也会在项目鸣谢中标注贡献者（以贡献者在 PR 或 Issue 中确认的公开名称为准）。

欢迎把你的成电记忆、校区地标、玩法创意和技术改进带回来。提交前请在 PR 或 Issue 中说明希望展示的姓名、届别或 GitHub ID；请勿提交自己或他人的敏感个人信息。
