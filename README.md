# 银杏神枪手 · 成电70年

电子科技大学 70 周年校庆主题的网页射击小游戏：在沙河与清水河校区地图中游园射击，收集校庆回忆与成电 AI 前沿目标。

> 这是校友共建的非官方创作项目，不代表电子科技大学官方立场。

## 在线体验

- https://www.bixinai.cn/Innovation/Gingko-Gunner/

## 本地运行

这是一个静态网页项目。克隆后使用任意静态文件服务打开根目录即可，例如：

```bash
npx serve .
```

然后访问输出的本地地址。

## 共建方式

欢迎校友提交 Issue 和 Pull Request：

- 补充两个校区的地标、建筑与回忆
- 改进移动端操作、性能与无障碍体验
- 设计目标物、音效、剧情和成就
- 校正学校历史、学院与校友企业相关资料

提交前请不要加入账号、密码、令牌、私钥或未经授权的图片/音频资源。
